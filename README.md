# minima-interact
Simple and powerful interface to interact with Minima blockchain using MiniMask including payments messaging and decentralized tools
## Features
- MiniMask detection
- Wallet connection status
- Address checking
- Balance checking
- Send Minima transaction
- Community message board interface
- Mobile responsive design
- Clean and professional UI
